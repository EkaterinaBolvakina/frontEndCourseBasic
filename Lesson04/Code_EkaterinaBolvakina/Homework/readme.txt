Homework 04:
-------------- EN ----------------
Build a page with 3 blocks (create new index.html and styles.css files)
Requirements:
- The blocks should contain user information:
    - first name last name
    - age
    - gender
- Blocks should meet the following requirements:
    - width 350px
    - height 300px
    - borders 2px
    - border style for each block is different (your choice)
    - internal indentation 20px
    - indents between cards should be 30px
- The size of the text inside the cards (content), choose such that the content does not pop out

-------------- DE ----------------
Eine Seite mit 3 Blöcken erstellen (neue index.html und styles.css Dateien erstellen)
Anforderungen:
- Die Blöcke sollten Benutzerinformationen enthalten:
    - Vorname Nachname
    - Alter
    - Geschlecht
- Die Blöcke sollten die folgenden Anforderungen erfüllen:
    - Breite 400px
    - Höhe 500px
    - Ränder 2px
    - Rahmenstil für jeden Block unterschiedlich (Ihre Wahl)
    - interner Einzug 20px
    - Einzüge zwischen den Karten sollten 30px betragen
- Die Größe des Textes innerhalb der Karten (Inhalt), wählen Sie so, dass der Inhalt nicht herausspringen
