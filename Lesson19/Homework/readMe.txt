Homework Lesson 19 
Task
1) Create a base class Transport representing a vehicle. 
a) The vehicle must have the following properties:
make
model
year
b) Add a displayMainInfo() method to the Transport class that displays all the information about the vehicle in the console.

2) Create two subclasses that extend the Transport class:
(a) Car - represents a vehicle. Additional properties:
numDoors (number of doors)
isConvertible (whether it is a convertible).
b) Motorcycle - represents a motorbike. Additional properties:
numWheels (number of wheels)
hasSideCar (whether it has a sidecar).

Create two instances each for the Car and Motorcycle classes
To demonstrate the results, call the displayMainInfo() method for each instance (4 calls).

Aufgabe
1) Erstelle eine Basisklasse Transport, die ein Fahrzeug darstellt. 
a) Das Fahrzeug muss die folgenden Eigenschaften haben:
make (Marke)
model (Modell)
year (Jahr)
b) Fügen Sie der Klasse Transport eine Methode displayMainInfo() hinzu, die alle Informationen über das Fahrzeug in der Konsole anzeigt.

2) Erstelle zwei Unterklassen, die die Klasse Transport erweitern:
(a) Car - repräsentiert ein Fahrzeug. Zusätzliche Eigenschaften:
numDoors (Anzahl der Türen)
isConvertible (ob es ein Cabrio ist).
b) Motorcycle - stellt ein Motorrad dar. Zusätzliche Eigenschaften:
numWheels (Anzahl der Räder)
hasSideCar (ob es einen Beiwagen hat).

Erstellen Sie jeweils zwei Instanzen für die Klassen Car und Motorcycle
Um die Ergebnisse zu demonstrieren, rufen Sie die Methode displayMainInfo() für jede Instanz auf (4 Aufrufe).

----------------------- RU ----------------------------
Задание
1) Создайте базовый класс Transport, представляющий транспортное средство. 
a) У транспортного средства должны быть следующие свойства:
make (марка)
model (модель)
year (год выпуска)
b) Добавьте в класс Transport метод displayMainInfo(), который выводит всю информацию о транспортном средстве в консоль.

2) Создайте два подкласса, расширяющих класс Transport:
a) Car - представляет автомобиль. Дополнительные свойства:
numDoors (количество дверей)
isConvertible (является ли кабриолетом)
b) Motorcycle - представляет мотоцикл. Дополнительные свойства:
numWheels (количество колес)
hasSideCar (имеет ли боковую коляску)

Создайте по два экземпляра для классов Car и Motorcycle
Для демонстрации результатов вызовите метод displayMainInfo() для каждого экземпляра (4 вызова).
