Lesson14 - Homework
Task
Suppose you have received an object with data (create a variable to store the object in) - spaceship, 
which contains the characteristics of the spaceship, such as:
name
manufacturer
crew
maxSpeed
You need to display them on the page.
Styling at your discretion
Additional task: Create a button "Hide Manufacturer", which will hide the manufacturer ship on the page.

----------------- DE ---------------------
Aufgabe
Angenommen, Sie haben ein Objekt mit Daten erhalten (erstellen Sie eine Variable, um das Objekt darin zu speichern) - Raumschiff, 
das Merkmale des Raumschiffs enthält, wie z. B:
Name
Hersteller
Besatzung
maxGeschwindigkeit
Sie müssen sie auf der Seite anzeigen.
Styling nach eigenem Ermessen
Zusätzliche Aufgabe: Erstellen Sie eine Schaltfläche "Hersteller ausblenden", die das Herstellerschiff auf der Seite ausblendet.

----------------- RU ---------------------
Задание

в котором находятся характеристики космического корабля, такие как:
name
manufacturer
crew
maxSpeed
Необходимо отобразить их на странице.
Стилизация на ваше усмотрение
Дополнительная задача: Создайте кнопку "Hide Manufacturer", которая будет скрывать manufacturer корабля на странице
