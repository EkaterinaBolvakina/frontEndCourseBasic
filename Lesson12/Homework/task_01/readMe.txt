Homework Lesson 12:
Task 01:
Create a form consisting of login, password and repeat password and submit button
when clicking on submit button the verification is performed - 
if the password and repeat password match - then the user gets a message: data sent 
otherwise the message password and repeat password do not match is displayed

--------------- DE ----------------
Erstellen eines Formulars, das aus Login, Passwort und Passwortwiederholung und der Schaltfläche "Absenden" besteht. 
Beim Klicken auf die Schaltfläche "Absenden" wird die Überprüfung durchgeführt - 
wenn Passwort und Passwortwiederholung übereinstimmen, erhält der Benutzer die Meldung: Daten gesendet, 
andernfalls wird die Meldung Passwort und Passwortwiederholung stimmen nicht überein angezeigt.

--------------- RU ----------------
создать форму состоящую из логин, пароль и повторите пароль и кнопки отправить при нажатии на кнопку отправить 
производится праверка - если пароль и повторите пароль совпадают - 
тогда пользователю выводится сообщение: данные отправлены 
иначе выводится сообщение пароль и повторите пароль не совпадают
