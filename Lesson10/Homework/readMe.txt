Homework lesson 10:
Task 1
Write a function that takes two numbers and returns the larger value. 
Assign the function call to a variable and display the result (variable) in the console

Task 2
Write a function that takes a string and a number n, and then returns the first n characters of the string. 
To display the result of the function, call it three times with different parameters, 
assign the result of each call to a different variable and display the variables in the console.

Task 3
Write a function that takes an array and an element, 
and then returns true if the element is present in the array, 
and false otherwise (if there are repetitions of a given element in the array, 
it does not affect the task). Display the result of the function call in the console

Task4* :
There is an array of numbers - sort it by the bubble method.

Aufgabe 1
Schreibe eine Funktion, die zwei Zahlen annimmt und den größeren Wert zurückgibt. 
Weisen Sie den Funktionsaufruf einer Variablen zu und geben Sie das Ergebnis (Variable) in der Konsole aus

Aufgabe 2
Schreibe eine Funktion, die eine Zeichenkette und eine Zahl n annimmt und die ersten n Zeichen der Zeichenkette zurückgibt. 
Um das Ergebnis der Funktion anzuzeigen, rufe sie dreimal mit unterschiedlichen Parametern auf, 
weise das Ergebnis jedes Aufrufs einer anderen Variablen zu und zeige die Variablen in der Konsole an.

Aufgabe 3
Schreibe eine Funktion, die ein Array und ein Element annimmt und dann true zurückgibt, 
wenn das Element im Array vorhanden ist, und andernfalls false 
(wenn es Wiederholungen eines bestimmten Elements im Array gibt, hat das keinen Einfluss auf die Aufgabe). 
Zeigen Sie das Ergebnis des Funktionsaufrufs in der Konsole an
