Homework Lesson 22
Task
You need to create a button "Get Cat Fact", when you click on it, a GET request will be executed (use fetch). 
The response to the request will be a random fact about cats
url:
https://catfact.ninja/fact
After you get a successful response, post the random cat fact on your page.
If the query ends in an error, it should be placed on the page and highlighted in red.
Each time the button is clicked, a new query should run and a new fact about cats should arrive.
Stylise it to your liking

------------------------- DE ----------------------------
Aufgabe
Sie müssen eine Schaltfläche "Get Cat Fact" erstellen. Wenn Sie darauf klicken, wird eine GET-Anfrage ausgeführt (verwenden Sie fetch). 
Die Antwort auf die Anfrage wird ein zufälliger Fakt über Katzen sein
url:
https://catfact.ninja/fact
Nachdem Sie eine erfolgreiche Antwort erhalten haben, veröffentlichen Sie die zufällige Katzeninformation auf Ihrer Seite.
Wenn die Abfrage mit einem Fehler endet, sollte dieser auf der Seite platziert und rot hervorgehoben werden.
Jedes Mal, wenn die Schaltfläche angeklickt wird, sollte eine neue Abfrage laufen und eine neue Tatsache über Katzen erscheinen.
Gestalten Sie es nach Ihrem Geschmack

------------------------- RU ----------------------------
Задание
Нужно создать кнопку "Get Cat Fact", при клике на которую, будет выполняться GET запрос(используйте fetch). 
В ответе на запрос будет приходить случайный факт о котах
url:
https://catfact.ninja/fact
После того как вы получите успешный ответ, разместите случайный факт о котах у себя на странице.
Если запрос завершиться ошибкой, её нужно разместить на странице и выделить красным цветом.
Каждый раз, когда происходит клик на кнопку, должен выполняться новый запрос и приходить новый факт о котах.
Стилизуйте на ваше усмотрение
