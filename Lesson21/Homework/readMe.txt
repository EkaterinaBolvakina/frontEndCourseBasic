Homework Lesson 21:
Task
Create a function 'delay' that takes a number (value) and returns a promis. 
The promis should resolve after 5 seconds and return double the value of the argument value (in resolve). 
If the argument is not a number, the promis should be rejected with the error text "The argument is not a number" (in reject)

----------------- DE ------------------
Aufgabe
Erstelle eine Funktion 'delay', die eine Zahl (value) annimmt und eine Promis zurückgibt. 
Die Promis soll nach 5 Sekunden aufgelöst werden und den doppelten Wert des Arguments value zurückgeben (in resolve). 
Wenn das Argument keine Zahl ist, soll die Promis mit dem Fehlertext "The argument is not a number" zurückgewiesen werden (in reject)

----------------- RU ------------------
Задание
Создайте функцию delay, которая принимает число (value) и возвращает промис. 
Промис должен разрешиться через 5 секунд и вернуть удвоенное значение аргумента value (в resolve). 
Если аргумент не является числом, промис должен быть отклонен с текстом ошибки "The argument is not a number" (в reject)
