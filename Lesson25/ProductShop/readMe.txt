Task: Implement a product page

Documentation:
https://dummyjson.com/docs/products

1) display all products on the page (layout at your discretion)
2) * On the page display both products and categories and 
   when clicking on a category display products of that category only.

3) Implement a page for adding a product 
on the page to display a form with the necessary fields and the submit button.
p.s. - public api https://dummyjson.com/docs/products does not give the possibility of real saving of new records. 
to the database, it only does an imitation of data saving...

4) Realise the page of displaying information on one product - on this page implement a carousel with pictures
 - add a button to delete the product (this api does not really delete the product - but does only an imitation of deleting the product).

5) On the main page with products and categories - implement a search by product name ie. 
on the page should be input with the type of text and when entering a value in it and pressing the button find 
- search for the product by name
examples url:
https://dummyjson.com/products/search?q=phone
https://dummyjson.com/products/search?q=laptop
https://dummyjson.com/products/search?q=lap

Aufgabe: Implementierung einer Produktseite

Dokumentation:
https://dummyjson.com/docs/products

1) Anzeige aller Produkte auf der Seite (Layout nach eigenem Ermessen)
2) * Auf der Seite sowohl Produkte als auch Kategorien anzeigen und 
   wenn Sie auf eine Kategorie klicken, werden nur die Produkte dieser Kategorie angezeigt.

3) Implementieren Sie eine Seite zum Hinzufügen eines Produkts 
auf der Seite ein Formular mit den notwendigen Feldern und dem Submit-Button anzeigen.
p.s. - die öffentliche API https://dummyjson.com/docs/products bietet keine Möglichkeit, neue Datensätze wirklich zu speichern. 
in der Datenbank, sondern nur eine Imitation der Datenspeicherung...

4) Realisieren Sie die Seite zur Anzeige von Informationen über ein Produkt - implementieren Sie auf dieser Seite ein Karussell mit Bildern
 - Fügen Sie eine Schaltfläche zum Löschen des Produkts hinzu (diese API löscht das Produkt nicht wirklich - sondern imitiert nur das Löschen des Produkts).

5) Auf der Hauptseite mit Produkten und Kategorien - implementieren Sie eine Suche nach Produktnamen d.h. 
auf der Seite sollte die Eingabe mit der Art von Text und bei der Eingabe eines Wertes in sie und drücken Sie die Schaltfläche finden 
- Suche nach dem Produkt nach dem Namen
Beispiele url:
https://dummyjson.com/products/search?q=phone
https://dummyjson.com/products/search?q=laptop
https://dummyjson.com/products/search?q=lap

документация:
https://dummyjson.com/docs/products

1) на странице отобразить все продукты (верстка на ваше усмотрение)

   при клике на категорию отображать продукты только данной категории

3) Реализовать страницу добвления товара

 - добавить кнопку удалить товар ( данное апи не дает реально удалить товар - а делает лишь эмитацию удаления товара)

на странице должен быть input с типом текст и при вводе в него значения и нажатия кнопки найти
